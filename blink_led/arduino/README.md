# Arduino Blink LED

Arduino 보드에서 LED를 깜빡이는 프로젝트입니다.

## 하드웨어 요구사항

- Arduino Uno/Mega/Nano 보드
- USB 케이블

## 개발 환경

- Arduino IDE 1.8.x 이상 또는 Arduino IDE 2.x
- Arduino AVR 보드 패키지

## 업로드 방법

1. Arduino IDE에서 `blink_led.ino` 파일 열기
2. 보드 선택: Tools > Board > Arduino Uno (또는 사용하는 보드)
3. 포트 선택: Tools > Port > (해당 포트)
4. 업로드 버튼 클릭 (Ctrl+U)

## 핀 설정

- LED 핀: 13번 핀 (내장 LED)
